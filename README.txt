You can run this program by running the AmongOS.exe file in the AmongOS folder. Alternatively you can start the program from Visual Studio.

To run the program from Visual Studio:
1. Open the AmongOS.sln file in Visual Studio.
2. Press F5 to run the program with debugger or Ctrl + F5 to run the program without debugger.

The entry class file that contains the main loop is main.cpp under the "Source Files" filter in the Solution Explorer.
